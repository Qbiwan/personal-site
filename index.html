How's it going

Hi! Welcome to Qbiwan's homepage.

In mid 2016, at the very tender age of 45, I decided to pick up a new skill, and started learning how to code from the internet. So I started learning R, then Python.

Thereafter I got interested in machine-learning and deep-learning, so I started taking online courses. These included Andrew Ng's deep learning and machine learning courses on Coursera, as well as fastai's practical deep learning for coders.
